# Snake Game Solver

A Python snake game solver and visualizer.

- Nearly 100% success rate.
- Average move count for 32x32 boards is ~66,000.

https://github.com/user-attachments/assets/7bc828e9-c266-4833-8fa0-fee5877d4bbb

## Usage

Interactive visualization:

```bash
python app.py solver
```

headless run:

```bash
python app.py solver --fast
```

Optional seed for reproducibility:

Press and hold Right arrow in the UI to advance steps (auto-repeat after hold).
